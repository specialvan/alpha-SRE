# Codex Run Card

## Mission

Build the standalone narrative SRE control plane that can later integrate with `alpha-autopilot`.

## Immediate priorities

1. Keep governance documents authoritative.
2. Define the replay and consistency model before implementation.
3. Make all changes reversible and reviewable.
4. Keep integration plans separate from current baseline control.

## Working rules

- One task at a time.
- Classify the task before coding.
- Write tests for any new behavior.
- Update the package index when documents move.
- Do not blur baseline and experimental work.

## Exit criteria

- The task is documented.
- The task is tested.
- The task is reversible.
- The task is placed in the correct layer.
