# Experimental README

`experimental/` stores draft ideas that are intentionally isolated from the baseline operating path.

## Rules

- Experimental work must not redefine baseline guarantees.
- Every experimental item should declare the hypothesis it is testing.
- Promote an item only after its contract, evidence, and rollback impact are documented.

## Minimum metadata for each experiment

- experiment name
- hypothesis
- affected contracts
- expected evidence
- exit condition
